# Predictive Analytics Using Historical Sales Data

## Project Overview
This project focuses on analyzing historical sales data and predicting future sales trends using Power BI. The dashboard provides meaningful business insights through data visualization, forecasting, and KPI analysis.

The main objective of this project is to identify sales patterns, understand customer and regional performance, and forecast future sales using historical data.

---

## Objectives
- Analyze historical sales performance
- Identify trends and business patterns
- Predict future sales using forecasting techniques
- Create an interactive Power BI dashboard
- Provide data-driven business insights

---

## Tools & Technologies Used
- Power BI
- CSV Dataset
- Data Visualization
- Forecast Analytics

---

## Dataset Information
The dataset contains historical sales records with the following fields:

- Order Date
- Sales
- Customer ID
- Category
- Segment
- Region
- State
- City
- Product Name
- Ship Mode

---

## Dashboard Features

### KPI Cards
- Total Sales
- Total Orders
- Total Customers

### Visualizations
- Sales Forecast Trend
- Category-wise Sales Analysis
- Region-wise Sales Distribution
- Segment-wise Customer Analysis

### Forecasting
Power BI forecasting was used to predict future sales trends based on historical sales data.

---

## Key Insights
- Technology category generated high sales performance
- West region contributed the highest sales
- Consumer segment had the largest customer share
- Forecast analysis shows future sales trend patterns

---

## Expected Outcome
This project helps businesses understand historical sales behavior and make better future business decisions using predictive analytics techniques.

---

## Conclusion
The dashboard successfully demonstrates how historical sales data can be transformed into actionable business insights using Power BI forecasting and visualization tools.
